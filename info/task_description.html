Enunciado

Joana, uma pesquisadora em biologia, está analisando sequências de DNA para detectar repetições consecutivas de nucleotídeos que podem indicar certas características genéticas. Ela precisa de uma função que identifique se uma sequência tem repetições consecutivas de um mesmo elemento. Escreva uma função que receba uma lista de elementos e retorne True se houver pelo menos uma repetição consecutiva do mesmo elemento, e False caso contrário.

Entrada: Uma lista de elementos.

Saída: Um valor booleano (True ou False).
